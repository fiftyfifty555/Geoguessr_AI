## Geoguessr
GeoGuessr - это игра о географических исследованиях. Цель игры - указать своё
местоположение в Google Street View. Победителем считается тот игрок, который
окажется ближе к загаданной точке. В ходе игры можно осматривать окрестности,
увеличивать изображение и двигаться по пути автомобиля на местных улицах.

Цели и задачи
Целями работы являются разработка и обучение нейронной сети, способной
распознавать любую улицу/ переулок/ шоссе в мире, используя базы данных Google
## Maps.
## Задачи:
1.Разобраться с работой и устройством нейронных сетей.
2.Рассмотреть различные методы и способы обучения нейронных сетей.
3.Реализовать программу, использующую один из способов обучения нейронных сетей и проверить
её работоспособность.

## Актуальность
Данный проект может быть полезен в правоохранительной (распознавание
местонахождения преступника по фото), информационной(определение
местоположения по картинке), киберспортивной сферах.

Способы машинного обучения
●Обучение с учителем
●Обучение без учителя
●Обучение с частичным привлечением учителя
●Обучение с подкреплением
●Глубинное обучение
